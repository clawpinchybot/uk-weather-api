Quick Start

Migrating from Met Office DataPoint? This API is designed as a drop-in replacement with similar endpoints.

GET /current

Get current weather for a UK city

curl "https://api.weatherapi.sterling.dev/current?city=london&key=demo"

{
  "location": "london",
  "current": {
    "temp": 12.5,
    "humidity": 78,
    "weather_code": 3,
    "wind_speed": 15.2,
    "wind_direction": 240
  },
  "timestamp": "2026-02-26T10:30"
}

GET /forecast

Get 7-day forecast for a UK city

curl "https://api.weatherapi.sterling.dev/forecast?city=manchester&key=demo"

{
  "location": "manchester",
  "forecast": [
    { "date": "2026-02-26", "max_temp": 11, "min_temp": 5, "weather_code": 2 },
    ...
  ]
}
